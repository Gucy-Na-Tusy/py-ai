# YOLOv5 Object Detection 

## Опис проєкту
Програма для детекції об'єктів у реальному часі з вебкамери з використанням YOLOv5.
Інтерфейс та назви об'єктів реалізовані українською мовою.

---

## Виконані завдання
- Вивід українських назв об'єктів
- Переклад класів YOLOv5 українською мовою
- Логування історії детекцій у файл
- Побудова траєкторій руху об'єктів
- Виділення ROI мишею
- Звукове сповіщення при появі важливих об'єктів
- Збереження скріншотів


> Примітка: OpenCV не підтримує кирилицю у cv2.putText :(

---

## Як запустити програму

1. Встановити Python 3.9+
2. Встановити залежності:
>pip install opencv-python torch numpy playsound pandas
3. Запуcтити файл:
>python homework.py

## Керування 

#### Керування з клавіатури
- Q — вихід з програми
- S — зберегти скріншот
- +/- — змінити поріг впевненості
- [ / ] — змінити мінімальний розмір об'єкта

### Керування мишею
- Затиснути ліву кнопку миші — початок ROI
- Відпустити кнопку — кінець ROI

## Приклади використання
- Детекція людей та автомобілів у реальному часі
- Аналіз руху об'єктів за допомогою траєкторій
- Обмеження детекції лише вибраною зоною (ROI)
- Збереження історії подій у файл detections.txt
- Звукове сповіщення при появі важливих об’єктів
- Збереження скріншотів у папку screenshots/
